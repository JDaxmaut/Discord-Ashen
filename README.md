## Features

- Deep neutral palette -- pure grey tones, zero warmth, no burgundy
- Gold accent -- `#f7bd48` used sparingly for interactive elements, badges, highlights
- Ultra-thin borders -- `#1a1a1a` separators throughout
- Custom scrollbars -- subtle `#353535` thumb, darkens on hover
- Full coverage -- every Discord surface, modal, popout, context menu, tooltip, call UI
- Call controls -- black action buttons with colour-coded icons (red for end, gold for answer)

## Installation

### BetterDiscord
1. Download `RetroPlayer.theme.css` from Releases
2. Move to `%appdata%/BetterDiscord/themes/`
3. Enable in BetterDiscord settings > Themes

### Vencord
1. Download `RetroPlayer.theme.css`
2. Move to `%appdata%/Vencord/themes/`
3. Enable in Vencord settings > Themes

### Manual
- Paste the CSS into your custom CSS field
