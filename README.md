## ✨ Features

- **Deep neutral palette** — pure grey tones, zero warmth, no burgundy
- **Gold accent** — `#f7bd48` used sparingly for interactive elements, badges, and highlights
- **Ultra-thin borders** — `#1a1a1a` separators throughout
- **Custom scrollbars** — subtle `#353535` thumb, darkens on hover
- **Full coverage** — every Discord surface, modal, popout, context menu, tooltip, and call UI
- **Call controls** — black action buttons with colour-coded icons (red for end, gold for answer)

## 📦 Installation

### BetterDiscord
1. Download [`RetroPlayer.theme.css`](https://github.com/JDaxmaut/Discord-Ashen/releases/latest/download/RetroPlayer.theme.css)
2. Move it to `%appdata%/BetterDiscord/themes/`
3. Enable in BetterDiscord settings > Themes

### Vencord
1. Download [`RetroPlayer.theme.css`](https://github.com/JDaxmaut/Discord-Ashen/releases/latest/download/RetroPlayer.theme.css)
2. Move it to `%appdata%/Vencord/themes/`
3. Enable in Vencord settings > Themes

### Manual (any client mod)
- Paste the CSS contents into your custom CSS field
